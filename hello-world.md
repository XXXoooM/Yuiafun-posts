## Hello World! 🌸

欢迎来到我的全新博客，由 [Astro](https://astro.build) 驱动，采用 [Moefun](https://github.com/XXXoooM/Yuiafun) 主题。

## 功能特性

这个博客拥有丰富的功能：

- ✨ **亮/暗模式** — 无缝主题切换，流畅过渡
- 📱 **响应式设计** — 在所有设备上都表现出色
- 🔍 **搜索** — Pagefind 支持的全文搜索
- 📑 **目录** — 文章内轻松导航
- 🏷️ **标签和分类** — 组织你的内容
- 📡 **RSS 订阅** — 订阅更新

## 为什么选择 Astro？

Astro 是一个现代 Web 框架，提供极致性能：

1. **默认零 JavaScript** — 只发送你需要的
2. **内容优先** — 完美适合内容驱动的网站
3. **群岛架构** — 部分水合，性能最优
4. **框架无关** — 使用 React、Vue、Svelte 或纯 HTML

## 代码示例

这是一个简单的 Astro 组件：

```astro
---
const greeting = "Hello, World!";
---

<h1>{greeting}</h1>
<style>
  h1 {
    color: purple;
  }
</style>
```
